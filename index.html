How to Create a Form on Cloudflare Pages Using FabForm.io

Overview

This tutorial will guide you through creating a simple HTML form hosted on Cloudflare Pages and using FabForm.io to handle form submissions.

Prerequisites

  • A Cloudflare account
  • A GitHub repository (to link with Cloudflare Pages)
  • A FabForm.io account

Step 1: Create a FabForm.io Form Endpoint

  1. Go to FabForm.io and sign in.
  2. Click "Create Form" and give it a name (e.g., "Contact Form").
  3. Copy the form endpoint URL (e.g., https://fabform.io/f/{your-form-id}).

Step 2: Create an HTML Form

Now, let's create a simple form that will submit data to FabForm.io.

1. Create a GitHub Repository

  • Go to GitHub and create a new repository (e.g., cloudflare-form).
  • Clone it to your local machine using:
    git clone https://github.com/your-username/cloudflare-form.git
cd cloudflare-form

2. Add the HTML Form

Inside the repository, create an index.html file and add the following code:

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Contact Form</title>
    <style>
        body {
            font-family: Arial, sans-serif;
            max-width: 600px;
            margin: 50px auto;
            padding: 20px;
            border: 1px solid #ddd;
            border-radius: 8px;
        }
        input, textarea, button {
            display: block;
            width: 100%;
            margin-top: 10px;
            padding: 8px;
        }
    </style>
</head>
<body>

    <h2>Contact Us</h2>
    <form action="https://fabform.io/f/{your-form-id}" method="POST">
        <label for="name">Name:</label>
        <input type="text" id="name" name="name" required>

        <label for="email">Email:</label>
        <input type="email" id="email" name="email" required>

        <label for="message">Message:</label>
        <textarea id="message" name="message" required></textarea>

        <button type="submit">Send</button>
    </form>

</body>
</html>

1. Push the Code to GitHub

Step 4: Test the Form

  • Open the URL and fill out the form.
  • Submit it and check FabForm.io for the submitted data.

Bonus: Add Form Validation & Success Message

If you want to add a success message, modify the form like this:

<form action="https://fabform.io/f/{your-form-id}" method="POST">
    <input type="hidden" name="_redirect" value="thank-you.html">
    <!-- Form fields -->
</form>
